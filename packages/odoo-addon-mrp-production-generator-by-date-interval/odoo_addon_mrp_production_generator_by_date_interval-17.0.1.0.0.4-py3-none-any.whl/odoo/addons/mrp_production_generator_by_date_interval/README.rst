This module allows planning multiple manufacturing orders in a single
action using a wizard that lets you define a period during which the
orders will be created and at what time they should be scheduled, based
on a product and a bill of materials.

**Table of contents**

.. contents::
   :local:

Usage
=====

1. Go to *Manufacturing > Planning > Production Generator*
2. Select the product and the BoM to be used
3. Select the period during which the orders should be created
4. Select the time at which the orders should be scheduled
5. Press **Create Productions**

You will see all the production orders created for each day

Known issues / Roadmap
======================

This module does not take into account that more than one MO may be
generated per day.
