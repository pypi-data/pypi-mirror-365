Support
=======

If you meet any issue or have any feature request, please send it to the support address `support@phoneshift.ing <support@phoneshift.ing>`_.

If you meet a bug, please join to your email:

* A minimal code snippet that reproduces the issue.

* The output of the following commands:

.. code-block:: shell

    export PHONESHIFT_API_KEY=api__valid_or_invalid_license_token_
    python -c 'import phoneshift; print(phoneshift.info)'
